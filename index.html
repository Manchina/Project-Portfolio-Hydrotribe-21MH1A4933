Voting App.

"A MERN stack application that is designed for casting the votes online without maintaining of other human. Initially, terms and conditions page will open, in which the user supposes to check the unput box. Later on the scanning page renders which takes a barcode present on the ID card and can cast their vote. This has react as frontend, Express js as the server side framework, Node runtime environment and MySql as the Database."
